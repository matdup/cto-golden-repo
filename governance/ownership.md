# Ownership Model

Defines accountability across the repository.

---

## Ownership Domains

| Domain | Owner | Backup |
|--------|-------|--------|
| Backend | Lead Backend | CTO |
| Frontend | Lead Frontend | CTO |
| Infrastructure | DevOps | CTO |
| Platform | DevOps | CTO |
| Contracts | Backend Lead | CTO |
| Governance | CTO | N/A |

---

## CODEOWNERS Alignment

CODEOWNERS must reflect:
- apps/backend → backend owner
- apps/frontend → frontend owner
- apps/infrastructure → infra owner
- governance → CTO

---

## Responsibilities

Owners are responsible for:
- Code review
- Security validation
- Dependency updates
- Incident response participation
- ADR review

Ownership implies accountability.
