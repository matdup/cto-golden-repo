# ADRs (Architecture Decision Records)

ADRs capture decisions that are:
- hard to reverse
- high-impact (security, correctness, cost, compliance)
- important for future maintainers

## Format
Each ADR includes:
- Context
- Decision
- Consequences
- Alternatives considered
- Links
