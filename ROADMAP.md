# Technical Roadmap

## Phase 0 — MVP
- Docker Compose runtime
- Single region
- Managed Auth
- Basic monitoring
- Daily backups

## Phase 1 — Growth
- Terraform multi-cloud abstraction
- Redis activation
- SLO alerts
- Restore drill automation
- Contract testing enforcement

## Phase 2 — Scale
- Managed Kubernetes
- Vault integration
- Per-context DB separation
- Advanced SLO burn-rate alerts
- WAF activation if risk justifies

---

Guiding principle:
Scale only when usage or organization requires it.
